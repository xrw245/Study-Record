# 第二章：变量和简单的数据类型

## 变量
* 每个变量指向一个值，且变量的值可以随时修改，程序记录最新的值。
1. 变量的命名和使用：
- 变量名只能包含字母、数字和下划线，且不能以数字为开头。
- 变量名不能包含空格
- 不能使用关键字和函数名

## 字符串
* 使用title()、upper()、lower()函数，对字符串进行首字母大写、都大写、都小写的操作，目的：将大小写单词视为同一个单词。
* f"{   }":在字符串中使用变量的值。(f字符串)

## 使用制表符或换行符来添加空白
* 制表符：\t
* 换行符：\n

## 删除空白
* 删除左空白：lstrip()
* 删除右空白：rstrip()
* 删除左右空白：strip()

## 删除前后缀
* 删除前缀：removeprefix("***")
* 删除后缀：removesuffix("***)

## 数
1. 运算法则：
* 加：+
* 减：-
* 乘：*
* 除：/
* 乘方：**
2. 整数和浮点数：
* 将任意两个数整除，结果总是浮点数。
* 任何运算中，有一个操作数是浮点数，结果也总是浮点数。
3. 数中的下滑线：
14000000 = 14_000_000
4. 同时给多个变量赋值：
x,  y,  z   =   0,  0,  0
5. 常量：整个程序的生命周期内都保持不变的量。

# 第三章：列表简介
1. 列表：排列的元素，用[    ]表示。

## 访问列表元素
* 索引是从0开始，而不是1开始。
* 索引为-1，可以返回最后一个列表元素，由此类推，-2为第二个元素。

## 修改、添加和删除元素
* 添加：append,     insert.
* 删除：del(永久删除),      pop(将元素弹出，仍可赋值使用),      remove(删除对应元素)

注：remove只能删除第一个指定的值。

## 列表排序
* 永久更改：sort
* 暂时呈现：sorted
* 反向打印：reverse
* 列表长度：len

## 编列整个列表
for语句：注意缩进和冒号。

## 创建数值列表
* range函数以一定步长遍历一组数。

* min,  max,    sum,求出最小、最大、和。
